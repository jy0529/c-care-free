# C盘无忧 V1 产品文档

## 产品目标

为 Windows 用户提供一个安全、清晰、可解释的桌面清理工具，重点解决：

- 垃圾缓存占用
- 常见应用缓存残留
- 大文件难发现、难搬移

## V1 范围

- 垃圾清理
- 大文件搬家

## 页面结构

### 垃圾清理

- 左侧竖导航
- 顶栏磁盘概览
- 分类卡片列表
- 底部已选汇总与清理按钮

### 大文件搬家

- 顶栏磁盘概览
- 扫描参数（阈值 / 数量上限）
- 文件列表（含搬家建议）
- 打开位置 / 搬移操作

## 风险边界

- 不允许任意路径删除
- 系统关键目录不执行删除
- 中高风险项默认不自动勾选
- 所有清理动作必须确认
