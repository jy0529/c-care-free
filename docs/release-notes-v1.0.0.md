# C盘无忧 v1.0.0

首个正式版本发布。

## 亮点

- 启动即显示 C 盘空间概览
- 垃圾清理：白名单规则、风险分级、分类勾选清理
- 大文件搬家：多线程 Top-N 扫描、实时预览、智能搬家建议
- 原生 WPF 界面，浅色 Dashboard 设计

## 环境要求

- Windows 10/11 x64
- .NET 8 Runtime（从源码运行需 SDK）

## 快速开始

```powershell
git clone git@github.com:jy0529/c-care-free.git
cd c-care-free
.\run.ps1
```
